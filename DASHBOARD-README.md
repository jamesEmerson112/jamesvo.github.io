## ğŸ“ COCOMO Calculations

The dashboard calculates COCOMO estimates using the **Organic Model**:

### Formulas
```javascript
KLOC = Code Lines / 1000

// Organic model parameters: a=2.4, b=1.05, c=2.5, d=0.38
Effort (person-months) = 2.4 Ã— (KLOC)^1.05
Time (months) = 2.5 Ã— (Effort)^0.38
People = Effort / Time
Cost = Effort Ã— (Average Wage / 12)
```

### Model Types
- **Organic** (default): Small teams, well-understood problems
- **Semi-detached**: Medium complexity projects
- **Embedded**: High complexity, large teams

You can customize by changing the parameters in the code.

## ğŸ› Troubleshooting

**Dashboard shows error "Failed to load scc-metrics.json"**
- Ensure `scc-metrics.json` is in the same directory as the HTML file

- Check internet connection (Chart.js loaded from CDN)

- Verify `scc-metrics.json` has correct data
